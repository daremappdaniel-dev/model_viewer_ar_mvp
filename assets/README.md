# Assets 3D

Este directorio contiene los modelos 3D utilizados en la aplicación AR.

## Archivos Requeridos

- **arrow.glb**: Modelo 3D de una flecha direccional.
- **daremapp_logo.glb**: Modelo 3D del logo de DareMapp para POIs.

> Nota: Los archivos binarios reales (.glb) deben ser añadidos manualmente al clonar el repositorio o subidos mediante Git LFS si exceden el tamaño permitido.
